How to add new CFP

  1. Go to https://github.com/callForPapers/callForPapers.github.io
  2. Fork the repository to your local machine
  3. Open the `_data/conferences` directory
  4. Add a new *.json file. Replace * with the name of the conference
  5. Edit the following json to meet your needs: 
    {
  "name" : "ACCU",
  "conferenceStart" : "2016-04-19",
  "conferenceEnd" : "2016-04-23",
  "callForPapersEnd" : "2015-11-13",
  "url": "http://accu.org/index.php/conferences/accu_conference_2016/accu2016_call_for_sessions",
  "lang": "en",
  "location": "Bristol, United Kingdom",
  "tags": "programmin, testing, architecture and design, development process, analysis, patterns, project management"
}
  6. Save the file
  7. Commit your changes
  8. Push the changes to your repository
  9. Create a pull request either through the GUI, GitHub directory or tool of your choice
  10. Pat yourself on the back for a job well done
  11. Congratulations - you are now and Open Source contributor
If any of this is unclear, give us a shout and we'll be more than happy to help you out.
